# 🚀 Partiu Entrega

O **Partiu Entrega** é uma solução completa para gestão de entregas e logística, conectando estabelecimentos, entregadores e clientes finais. O projeto foca em oferecer uma experiência fluida, com atualizações em tempo real e uma interface intuitiva.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **Frontend:** [Ex: React.js, Next.js, Tailwind CSS]
- **Backend:** [Ex: Node.js, Express, TypeScript]
- **Banco de Dados:** [Ex: PostgreSQL, MongoDB]
- **Mobile:** [Ex: React Native, Expo]
- **Outras ferramentas:** [Ex: Docker, Prisma, Socket.io para tempo real]

## 📋 Funcionalidades

- **Dashboard do Estabelecimento:** Gestão de pedidos, status de preparo e histórico de vendas.
- **Painel do Entregador:** Visualização de rotas disponíveis, aceitação de corridas e integração com mapas.
- **Acompanhamento em Tempo Real:** Status do pedido atualizado instantaneamente para o cliente.
- **Autenticação Segura:** Sistema de login com JWT e diferentes níveis de acesso (Admin, Entregador, Cliente).
- **Interface Responsiva:** Design adaptado para dispositivos móveis e desktop.

## 🚀 Como Executar o Projeto

### Pré-requisitos
Antes de começar, você precisará ter instalado em sua máquina:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/).

### 🏁 Rodando a aplicação

```bash
# Clone este repositório
$ git clone [https://github.com/feliciocavalcante/partiu-entrega.git](https://github.com/feliciocavalcante/partiu-entrega.git)

# Acesse a pasta do projeto no terminal
$ cd partiu-entrega

# Instale as dependências
$ npm install

# Inicie o servidor (Backend)
$ npm run dev:server

# Inicie a aplicação (Frontend)
$ npm run start
